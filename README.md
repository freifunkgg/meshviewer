# Meshviewer

Meshviewer is an online visualization app to represent nodes and links on a map for Freifunk open mesh network.

This is a fork of https://github.com/ffrgb/meshviewer.

### Demo

Standalone: https://test.blauerbecher.de

## Dokumentation
### Install macOS
```
brew install npm
brew install yarn
yarn
```
### Bauen
```
gulp
```

### Lokal testen
```
tiny-server build
```
